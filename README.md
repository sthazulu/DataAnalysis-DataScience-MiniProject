# Customer Support Insights Analysis

A Data Science mini project showcasing skills developed so far during my data science bootcamp through the analysis of customer support data to identify trends in issues, resolution times, and customer satisfaction.

## Project Overview
This project explores a small dataset containing customer support tickets.  
The goal is to identify patterns that can be used to improve customer experience, issue management, and operational efficiency.

## What’s Inside the Project
- `customer_support_data.csv` — sample dataset  
- `customer_support_data_analysis.ipynb` — Python notebook with code, graphs, and insights  
- Visualisations created using Matplotlib  
- Summary of findings

## Tools Used
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook

## Key Insights
- Identified most common customer issues  
- Compared resolution times across issue categories  
- Analysed satisfaction rating trends  
- Highlighted opportunities for process improvement
